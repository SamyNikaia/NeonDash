# NeonDash

Jeu mobile iOS — one-tap reflex, esthétique neon / synthwave.

## Stack

- SwiftUI + SpriteKit
- iOS 17+
- Swift 5.9

## Build

Le projet Xcode est généré via [XcodeGen](https://github.com/yonaskolb/XcodeGen).

```sh
brew install xcodegen
xcodegen generate
open NeonDash.xcodeproj
```

## Gameplay

Tu contrôles une bille qui rebondit entre deux rails parallèles. Tap pour switcher de rail, esquive les obstacles, bats ton meilleur score.
